## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Create env

```bash
$ DATABASE_URL=
$ JWT_SECRET=
```

## Project setup

```bash
$ yarn install
```

## Prisma execute
```bash
$ npx prisma migrate dev --name init

$ npx prisma generate
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
